# Vibe coding — Google AI Studio

**How to use (above):**
1. Visit https://aistudio.google.com/apps
2. Click a model to open **Advanced settings** — you may choose a different model there as well.
3. Choose React or Angular according to your needs.
4. In **System instructions**, pick **Upload an instruction file** from the dropdown and upload the instruction file you downloaded.
